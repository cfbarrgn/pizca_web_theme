# Flow Theme — Pizca Chile

Custom Shopify theme based on Flow, managed via GitHub + Shopify CLI.

## Branches
- `main` — live production theme
- `dev` — active development branch

## Setup
1. Install Shopify CLI: `npm install -g @shopify/cli`
2. Authenticate: `shopify auth login`
3. Preview locally: `shopify theme dev --store your-store.myshopify.com`
